## 🎨 情感指令示例

### 古诗朗读
```bash
-i "用沉稳、富有感情的语气朗读古诗，注意停顿和韵律"
```

### 故事讲述
```bash
-i "用讲故事给小孩听的语气，生动有趣"
```

### 新闻播报
```bash
-i "用专业、清晰、中立的语气播报新闻"
```

### 安慰/哄睡
```bash
-i "用特别温柔、轻柔的语气，像在哄小孩睡觉"
```

### 激情演讲
```bash
-i "用激昂、充满激情的语气演讲"
```

### 幽默搞笑
```bash
-i "用幽默、轻松的语气，带点调侃的味道"
```

## 📊 参数调优

### Temperature（温度）
- `0.5-0.7`：稳重、标准
- `0.8`：**默认**，平衡
- `0.9-1.0`：活泼、多变

### Repetition Penalty（重复惩罚）
- `1.0-1.05`：允许重复
- `1.1`：**默认**，平衡
- `1.2-1.5`：严格避免重复

## 💡 使用技巧

### 1. 避免截断
默认会添加 0.5 秒尾部静音，如果不需要可以用 `--no-silence`

### 2. 批量生成
```bash
for text in "第一句" "第二句" "第三句"; do
  qwen-tts "$text" -q
done
```

### 3. 自定义输出
```bash
qwen-tts "你的文本" -o custom_name.mp3 -s Uncle_Fu -i "你的情感指令"
```
