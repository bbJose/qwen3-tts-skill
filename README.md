## 🎨 示例

### 古诗朗读（推荐）
```bash
qwen-tts "静夜思 - 李白。床前明月光，疑是地上霜。举头望明月，低头思故乡。" \
  -s Uncle_Fu \
  -i "用沉稳、富有感情的语气朗读古诗"
```

### 故事讲述
```bash
qwen-tts "从前有座山，山里有座庙..." \
  -s Vivian \
  -i "用讲故事给小孩听的语气"
```
